# ORM

### ORM 이란?
* Object Relational Mapping = 객체로 연결 해준다
* SQL 언어가 아닌 애플리케이션 개발 언어로 데이터베이스를 접근할 수 있게 해주는 툴
* ORM 은 SQL 문법 대신 애플리케이션의 개발 언어를 그대로 사용할 수 있게 함으로써, 개발 언어의 일관성과 가독성을 높여준다는 장점을 가지고 있음.
