
## 클래스 변수와 인스턴스 변수

* 클래스 변수 (static 변수) : 클래스별로 유일
* 인스턴스 변수 : 인스턴스가 소유하는 변수



### 순서

1. 상위 클래스 초기화
	* 객체가 상속받은 클래스가 있다면, 상위 클래스의 생성자가 먼저 호출됨.
	* 이 과정에서 상위 클래스의 멤버 변수들이 초기화 됨.
2. 멤버 변수 선언과 함께 초기화
	* 클래스 내에서 선언과 동시에 초기화된 멤버 변수들은 상위 클래스의 생성자 호출 직후, 하위 클래스의 생성자가 실행되기 전에 초기화됨.
