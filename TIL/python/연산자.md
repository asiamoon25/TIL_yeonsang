_연산자는 피연산자의 값을 조작할 수 있는 구조_

## 1. 산술 연산
* '+' : 덧셈(Addition)
```python
print(3 + 2 )
# Output : 5
```

* '-' : 뺄셈
```python
print(3-2)
# Output : 1
```

* '\*' 곱셈
```python
print(3*2)
# Output 6
```

* '/' 나눗셈
```python
print(3/2)
# Output 1.5
```

* '//' 정수 나눗셈
```python
print(3//2)
# Output : 1
```

* '%' 나머지
```python
print(3%2)
# Output : 1
```

* '\*\*' 거듭제곱
```python
print(3 ** 2)
# Output : 9
```
