## 개요
* 경마 경주 결과 통계
* 각 말들 승률(패널티 등.. 짐 같은거 지었을 때 등..)
* 가장 이상적인 배팅

위 3가지를 할 수 있는 사이트를 만들어야함.


## 데이터 수입처
* 공공데이터 포탈
	* 경마 관련 API
